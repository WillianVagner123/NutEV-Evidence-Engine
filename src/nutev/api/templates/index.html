NutEV Platform

Evidence Engine for Lifestyle Nutrition

Plataforma local para monitoramento, auditoria e tradução de evidências em recomendações candidatas para o Protocolo Dietético NutEV.

Esta plataforma é local. Ela não aprova recomendações finais e não substitui revisão humana.

  • Evidence Engine — Matriz global de evidências e lentes analíticas.
  • Audit Engine — Claims extraídos, status e rastreabilidade documental.
  • Recommendation Candidates — Recomendações candidatas vinculadas a claims.
  • Human Review Queue — Itens que precisam de validação humana.
  • Run Summary — Métricas da execução atual.
  • Artifacts — Lista de outputs gerados.
  • API Docs — Documentação interativa da API local.

Control Center: rode nutev dashboard --project-root ./project_output_demo --port 8501
