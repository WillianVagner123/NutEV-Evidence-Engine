# Changelog Metodológico

- 2026-05-21: Implementação inicial do NutEV Audit Engine para rastrear a cadeia documento → claim → recomendação candidata, reduzindo risco de recomendações não suportadas.
- 2026-07-21: Ampliação das fontes bibliográficas padrão da busca (Artigo 1 — revisão de escopo JBI/PRISMA-ScR). O conector DOAJ (Directory of Open Access Journals), já declarado no `source_priority` das buscas desde 2026-06-27, passou a ser efetivamente executado após a correção do despacho de provedores (o pipeline agora deriva os provedores suportados do registro implementado do orquestrador). Adicionado o conector SciELO ao `source_priority` padrão das buscas (busca1, busca2a, busca2b, artigo3_framework), recuperando literatura ibero-americana/brasileira sistematicamente sub-representada nos indexadores internacionais — reforço metodológico relevante para uma tese do PPGNH/UnB. Ambos operam por padrão em modo de página única (reprodutível); a recuperação ampliada continua sendo opt-in por variável de ambiente (`NUTEV_DOAJ_MAX_RESULTS` / `NUTEV_SCIELO_MAX_RESULTS`). A cobertura do SciELO é escopada ao prefixo DOI 10.1590 via API estável do Crossref (documentado em `docs/SEARCH_PROVIDERS.md`).
