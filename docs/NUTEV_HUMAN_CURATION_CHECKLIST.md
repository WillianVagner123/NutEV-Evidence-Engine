# NutEV Human Curation Checklist

- [ ] O documento é elegível para o protocolo?
- [ ] O tipo documental está correto?
- [ ] O domínio NutEV foi marcado corretamente?
- [ ] O claim é textual (extraído) ou inferência?
- [ ] Existe `exact_quote`?
- [ ] O `quote_location` está correto?
- [ ] A recomendação candidata tem `supporting_claim_ids`?
- [ ] Há conflito de evidência?
- [ ] Precisa de segundo revisor?
- [ ] Pode alimentar diretriz/pirâmide/competência/menu/adesão?
